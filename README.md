# Ant-Design-Vue
Attempts with the Table and Comment components of Ant Design Vue.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

## Operating steps
While complied successfully, open your browser on http://localhost:8080/.
